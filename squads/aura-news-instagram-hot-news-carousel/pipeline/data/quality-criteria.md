# Quality Criteria — Aura News Squad

## Research
- [ ] Cada claim central possui ao menos 2 fontes.
- [ ] Fontes primarias priorizadas e data registrada.
- [ ] Nivel de confianca atribuido para cada insight.

## Copy
- [ ] Hook inicial claro em ate 125 caracteres.
- [ ] Narrativa sem redundancia entre slides.
- [ ] CTA final especifico e acionavel.
- [ ] Linguagem acessivel para publico geral.

## Design
- [ ] Formato 1080x1440 consistente em todos os slides.
- [ ] Hierarquia visual clara e leitura instantanea.
- [ ] Contraste adequado para leitura em celular.
- [ ] Coesao visual do slide 1 ao slide final.

## Review
- [ ] Score geral >= 7/10.
- [ ] Nenhum criterio critico abaixo de 4/10.
- [ ] Feedback com correcao objetiva para itens reprovados.

## Publish
- [ ] Preview apresentado ao usuario antes do publish.
- [ ] Dry-run executado e aprovado.
- [ ] Confirmacao explicita do usuario registrada.
- [ ] Resultado final com URL/ID e timestamp.
